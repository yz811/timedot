Time Dots ⏳
看见时间的流逝。 一个极简主义、高可视化的桌面时间管理工具，基于 Python 与 PyQt6 构建。

Time Dots 不仅仅是一个待办事项列表。它将你的一天具象化为由“点”组成的网格。每一个点代表 10 分钟（可配置），每一行代表一小时。通过视觉化的方式，它帮助你对抗“时间盲区”，让时间的流逝变得可感知、可触摸。

✨ 主要特性
视觉化点阵系统：用点阵代替数字时钟，直观展示过去（灰色）、现在（高亮）与未来（白色）。

无边框悬浮设计：极简 UI，支持“呼吸”效果（鼠标悬停展开，移开收缩）。

时间块 (Time Blocking)：简单的拖拽即可创建时间段（Segment），用于规划专注工作或会议。

时间标记 (Time Note)：双击任意时间点添加备注，记录当下的瞬间。

穿透模式 (Lock Mode)：锁定后窗口背景透明并允许鼠标穿透，像水印一样浮在桌面上，不干扰正常工作。

高度可定制：支持调整点的大小、间距、每行时长、颜色主题以及字体。

底部日历：集成的迷你日历，支持查看过去或规划未来日期的日程。

声音反馈：(Windows Only) 计时结束或到达标记点时提供轻柔的提示音。

🎯 使用场景 (Use Cases)
Time Dots 专为以下场景设计：

1. 对抗“时间盲区” (ADHD 友好)
对于常常感觉不到时间流逝的人群，Time Dots 将抽象的时间转化为物理的“空间”。看着代表当前小时的行被一点点填满，能有效缓解拖延，建立时间紧迫感。

2. 深度工作与番茄钟
无需复杂的番茄钟软件。只需在点阵上拖拽出一段 30 分钟或 60 分钟的 Segment，它就成了一个可视化的进度条。当光标走到 Segment 终点时，你就知道休息时间到了。

3. 会议与日程概览
在屏幕一角常驻。当你在开会时，一眼就能看到距离会议结束还有多少个“点”，从而更好地把控会议节奏，避免超时。

4. 碎片化记录 (Intersticial Journaling)
利用 Note 功能，在工作的间隙快速标记此刻的状态（例如：“开始写代码”、“被打断”、“收到邮件”）。一天结束时，你拥有一份完整的时间由数据。

🎮 操作指南
Time Dots 采用了独特的交互方式以保持界面简洁：

基础交互：

左键拖拽：在点阵上拖拽以创建时间块（Segment）。

右键点击：点击时间块或标记点，弹出编辑/删除菜单。

滚轮滚动：在日历区域滚动切换日期。

控制区域（左上角悬停显示）：

🔴 红灯：退出程序。

🟡 黄灯：隐藏窗口（可通过托盘图标唤醒）。

🟢 绿灯：切换 锁定/穿透模式。

设置：

通过系统托盘图标右键 -> 设置，或在非锁定状态下右键菜单进入设置。

你可以自定义起始时间（如 9:00 - 18:00）、颜色主题及网格密度。

⚙️ 配置说明
所有配置会自动保存在同目录下的 config.json 文件中。支持自定义的内容包括：

Row Duration：每行代表的时长（30m, 1h, 2h 等）。

Interval：每个点代表的分钟数（5m, 10m, 15m）。

Theme：背景色、高亮色、过期时间颜色等。
